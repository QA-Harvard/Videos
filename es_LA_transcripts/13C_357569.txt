DOREEN:  Soy Doreen, Gerente de Distrito
 de una empresa de material médico. 
Implantamos un nuevo sistema 
de distribución que funcionaba muy bien.
 Pero una persona no opina lo mismo. 
Hola, Robert. 
ROBERT:  Hola, Doreen. 
DOREEN:  Hemos trabajado en este sistema
 y querría saber qué te parece, 
porque creo que no te convence mucho. 
ROBERT:  Sí, es un poco raro tener 
este sistema implantado. 
Me gustaba el sistema anterior, 
era bastante fácil de entender, 
pero el nuevo es muy complicado. 
Se supone que este nuevo sistema 
está diseñado para prevenir errores, 
pero veo que puede provocar más. 
No estoy muy conforme 
con la implantación del nuevo sistema. 
DOREEN:  Vale, entonces no te terminas 
de acostumbrar al nuevo sistema, 
porque estabas acostumbrado al antiguo 
y pensabas que funcionaba muy bien. 
y que este sistema tiene más errores, 
¿verdad? 
ROBERT:  Eso es, sí. 
DOREEN:  De acuerdo. 
Dame más información. 
¿Con qué parte del sistema 
tienes problemas exactamente? 
ROBERT:  El software es mucho más
técnico 
que el anterior que teníamos. 
Creo que el tiempo para comprender 
el software será mucho mayor. 
DOREEN:  Entonces crees que necesitas
algo 
más de tiempo para entender el software 
y manejarlo con más soltura. 
Necesito que intentes trabajar conmigo 
y le des al sistema algo más de tiempo 
para que te demuestre que puede 
funcionar todavía mejor que el anterior.
 ¿Crees que podrás hacerlo? 
Dame un tiempo para demostrártelo. 
ROBERT:  Sí, claro que puedo. 
Y muchas gracias por escucharme.
